# Taller BordeLago Website

Este paquete contiene la estructura base de la página web.

## Contenido
- index.html → Página principal con la estructura HTML/CSS completa.
- /images/ → Carpeta donde debes colocar tus imágenes.

## Instrucciones
1. Coloca tus imágenes reales en la carpeta /images/ con estos nombres:
   - hero.jpg
   - hero-product.jpg
   - taller.jpg
   - collares.jpg
   - tejidos.jpg
   - cojines.jpg
   - cuadros.jpg
   - mantitas.jpg
   - lamparas.jpg
   - espejos.jpg
   - birdhouse.jpg
   - extra1.jpg, extra2.jpg, extra3.jpg
2. Abre `index.html` en un navegador para ver la página.
3. Para publicarla en internet, sube la carpeta completa a un hosting (ej: Netlify, GitHub Pages, Vercel).

## Nota
El formulario de contacto está en modo demostración. Para enviar mensajes reales, necesitas conectar un servicio de backend o email.
